Afin d'installer l'application correctement,

1 installez un logiciel de base de données locale tel que Uwamp

2 créez une base de donnée nommée "onlineschool"

3 executez le script d'installation compris dans les fichiers (Onlineschool.sql)

4 Copiez les fichiers du site dans le dossier www de Uwamp

5 rendez-vous sur la page en local
